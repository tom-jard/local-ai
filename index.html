Get Started in 5 Minutes

  1. Go to ollama.com and click Download
  2. Open the app. It includes a built-in chat window, model downloads, and file drag-and-drop.
  3. Browse the model library in the app and download one. Start chatting.
  4. That's it. You're running AI on your machine. No internet needed after download.

Optional (advanced): Open Terminal and type ollama run <model-name> for a command-line interface.

  1. Go to ollama.com and click Download
  2. Run the installer, then open the Ollama app. It has a built-in chat window and model browser.
  3. Browse the model library in the app and download one. Start chatting.
  4. That's it. AI running locally on your PC.

Optional (advanced): Open Command Prompt and type ollama run <model-name> for a command-line interface.

  1. Open your terminal and run:
    curl -fsSL https://ollama.com/install.sh | sh
  2. Then start chatting:
    ollama run <model-name>

What to Try First

Tap any prompt to copy it. Paste into the Ollama app or terminal.

Explain [topic from my class] like I'm five
Help me outline an essay about climate change solutions
What are 3 ways I can reduce my energy use as a college student?
Proofread this paragraph and suggest improvements: [paste your text]
Create a study guide for [subject]
Write a cover letter for [job or internship]

Want to Go Further?

  • Prefer a different app? LM Studio is a polished desktop alternative with a built-in model browser. Same idea, different interface.
  • Try other models: Browse ollama.com/library for options sorted by size and capability
  • Try a bigger model: Larger models need more RAM but are noticeably smarter
  • Study with your course materials: copy-paste readings into the chat and ask questions about them
  • AI agent tools: OpenClaw (assistant workflows) or OpenCode (writing, planning, research, coding)
  • Browse all models: ollama.com/library
